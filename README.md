# vibe-class-web (바이브코딩 클래스)

## 이 프로젝트는?
- 누구나 쉽게 원데이 코딩 클래스를 신청/문의할 수 있는 "웹사이트"입니다.
- 학생이나 학부모가 들어와서, 수업소개를 보고, 직접 문의를 할 수 있게 만들어졌습니다.

## 폴더와 파일이 무슨 역할인가요?

```
vibe-class-web/
┣ public/        → 이미지나 로고, 파비콘(브라우저에 뜨는 작은그림) 등이 들어가는 곳입니다!
┣ src/           → 우리 웹사이트 진짜 내용을 만드는 핵심 공간입니다.
┃ ┣ App.js       → 화면에서 보이는 거의 모든 내용을 만드는 가장 중요한 파일! 탭, 문의, 폼 등 모두 여기에 작성됨
┃ ┣ App.css      → 화면에 보이는 디자인(색, 글씨, 위치, 버튼 스타일 등)을 담당합니다.
┃ ┣ index.js     → 실제 웹사이트 전체를 처음 띄울 때 참고하는 일종의 시작점 역할. 대부분 건드릴 필요 없음
┃ ┗ ...
┣ package.json   → 이 프로젝트에서 어떤 프로그램/패키지를 쓰는지 적어놓은 기록지! 거의 자동관리됨
┗ README.md      → (바로 이 파일!) 프로젝트 전체 설명서. 뭔지, 어떻게 쓰는지 적음
```

## 우리 웹사이트 주요 기능과 설명 (왕초보용)

- 상단 "메뉴(탭)"를 누르면 필요한 소개/정보만 바꿔서 볼 수 있음!
    - 예) 수업 구성, 만들 수 있는 것들, 비용/문의 등
- **문의하기 버튼**을 누르면 팝업창(모달)이 뜨고, 여기에 이름, 연락처, 질문을 입력해서 바로 문의할 수 있음.
- 문의를 보내면 바로 Google 스프레드시트(엑셀처럼 여러 줄로 문의가 쌓임)로 기록돼서, 선생님이 실시간으로 확인할 수 있음!

## 각 코드 파일에서는 (왕초보 설명)
- App.js : 기능 추가/수정/고장은 전부 여기서 관리! 문의폼, 탭, 버튼 등
- App.css : 색깔/폰트/버튼(예쁘게 보이게) 고칠 때 여기서 수정

## 개발자/운영자가 자주 하는 작업
- npm start → 웹사이트 실행(내 컴퓨터에서 테스트)
- 코드를 고칠 때마다(예: 문구 바꿀 때) App.js나 App.css를 저장하고, 여기 README도 간단 메모나 설명을 추가!
